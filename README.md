# Dashboard Aging Kelulusan 2026 Mobile

Versi ini diasingkan daripada folder `2026` asal dan direka untuk paparan mobile dengan dua tab sahaja:

1. Dashboard Kelulusan 5 Hari
2. Jadual Prestasi Keseluruhan

## Cara Guna

1. Buka `index.html` dalam browser.
2. Pilih fail CSV permohonan.
3. Data diproses sepenuhnya dalam browser pengguna.

## Keselamatan Data

- Jangan masukkan CSV atau Excel sebenar dalam repo GitHub.
- Versi ini tidak memaparkan nama pemohon, No KP, reference number, pelulus, atau peraku.
- Export hanya menghasilkan jadual agregat bulanan.
- Jika perlukan contoh data, gunakan data dummy/anonymized sahaja.

## Deployment GitHub Pages

Upload kandungan folder ini ke repo GitHub, kemudian aktifkan GitHub Pages. Fail CSV sebenar kekal di komputer pengguna dan dipilih secara lokal melalui browser.
